## Running locally

To run locally, you must first install node, npm, Ruby (we recommend RVM), ruby-gems, and the bundler.

Then, run:

```bash
npm install
bundle install
```

To download all the dependencies.

Then, run:

```bash
bundle exec jekyll serve
```

To start the local server.
