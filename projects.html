Our Projects

Blue Fire is home of many Flutter-adjacent projects and libraries, including:

  • Flame, a 2D game engine made on top of Flutter
  • Audioplayers, a Flutter plugin for playing native audio
  • Photo View, a customizable zoomable image widget for Flutter
  • Dashbook, some development tools to help you build UI on Flutter
  • Daylight, which computes the sunset and sunrise times for a geolocation
  • Fast Noise, a Dart port of classical noise algorithms
  • Canvas Test, a mocking utility for testing the Dart UI's Canvas
  • Ordered Set, a fast, reliable and queryable implementation of an ordered set for Dart
